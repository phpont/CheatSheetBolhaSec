Hackers Do Bem

Bem-vindo à Cheatsheet Hackers do Bem

Navegar pelo mundo do hacking pode ser uma jornada complexa e desafiadora, especialmente para quem está começando. Pensando nisso, criamos esta cheatsheet como uma bússola confiável para guiar sua aprendizagem. Aqui, você encontrará comandos essenciais, técnicas fundamentais, e dicas práticas que são pedras angulares no campo do hacking ético.

Seja você um estudante buscando aprimorar suas habilidades, um entusiasta querendo explorar novos horizontes, ou um profissional procurando um recurso rápido para consultas diárias, nosso objetivo é fornecer um conteúdo claro, direto e de fácil acesso. Utilize esta página como seu guia de referência rápida.

Comando ls

Lista todos os arquivos e diretórios.

Comando cd

Muda o diretório atual.

Comando pwd

Exibe o caminho completo do diretório atual.

Comando mkdir

Cria um novo diretório.

Comando rm

Remove arquivos ou diretórios.

Comando cp

Copia arquivos e diretórios.

Comando mv

Move ou renomeia arquivos e diretórios.

Comando cat

Exibe o conteúdo de um arquivo ou os concatena.

Comando chmod

Altera as permissões de arquivo ou diretório.

Comando man

Exibe o manual de um comando.

Comando touch

Cria um novo arquivo vazio ou atualiza a data de modificação de um arquivo existente.

Comando grep

Procura por padrões em arquivos ou entrada de dados. Muito útil para filtrar informações.

Comando find

Busca arquivos e diretórios com base em critérios de pesquisa específicos.

Comando tail

Exibe as últimas linhas de um arquivo. Comum para monitoramento de logs.

Comando head

Exibe as primeiras linhas de um arquivo.

Comando df

Mostra informações sobre o uso do espaço em disco dos sistemas de arquivos montados.

Comando top

Exibe informações em tempo real sobre processos em execução e uso de recursos do sistema.

Comando ps

Exibe informações sobre processos ativos. Pode ser combinado com grep para filtrar processos.

Comando kill

Envia sinais a processos. Geralmente usado para terminar processos.

Comando sudo

Permite executar comandos com privilégios de segurança elevados, geralmente como o superusuário.
